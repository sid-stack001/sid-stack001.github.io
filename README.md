A single-file interactive portfolio that boots like an operating system. Visitors explore content by clicking terminal commands in the sidebar, no scrolling, no sections, just a terminal that responds.

## Features

- **Pure HTML/CSS/JS** — zero frameworks, zero dependencies, single `index.html` file
- **Neural network background** — canvas-based particle system with mouse interaction
- **Terminal engine** — custom async command runner with typewriter output
- **Boot sequence** — simulated OS boot on page load
- **Playable Snake** — fully functional game embedded in the terminal
